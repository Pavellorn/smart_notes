# smart_notes

Smart Notes - это декстопное приложение для физиков-теоретиков для создания, редактирования, удаления заметок с поддержкой тегов и поиска.

![alt text]({837E4588-2CAF-406E-9FCB-4473AF4B1C98}.png)

Возможности:
- Создание заметок с текстом до 500 символов
- Добавление тегов к заметкам
- Поиск по тексту и тегам
- Редактирование существующих заметок
- Удаление заметок
- Автоматическое сохранение в JSON файл
- Автогенерация ID для заметок
- Отображение времени создания и обновления заметок
